# Memory, Arrays and Lists in Java

In this module, I studied how memory works in Java and how data structures such as arrays and lists are handled internally.
I also practiced these concepts through fixation exercises.

## Topics Covered
- Primitive types vs reference types
- Memory allocation (stack and heap)
- Garbage Collector behavior
- Arrays (one-dimensional and two-dimensional)
- Accessing and iterating over arrays
- Wrapper classes (boxing and unboxing)
- Enhanced for loop (for-each)
- Lists (`List`, `ArrayList`)
- Basic list operations (add, remove, search and filter)

## Exercises
- Average height using arrays
- Average product price using object arrays
- Room rental system using arrays
- Employee salary increase using lists and encapsulation
- Matrix operations (main diagonal and negative numbers)
- Matrix position and neighbors search

All exercises were implemented using Java and focused on understanding memory behavior and data structure manipulation.
